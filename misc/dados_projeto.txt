Precisa-se ter uma IA que faça com aprofundamento os temas requeridos(Ou seja, precisamos também criar um prompt que entregue um bom resultado, precisamos de prompt engineering)
Precisa-se de uma interface web responsiva. Web é melhor pois de web pode-se usar em qualquer lugar.

Penso que a melhor forma de usar IAs é com web scraping, pois os provedores de api de ia geralmente permitem o uso de modelos fracos ou então com limites muito limitantes.
